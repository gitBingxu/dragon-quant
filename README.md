龙头战法四维量化筛选 — 核心逻辑
整体流程
从当日涨停榜出发，过滤掉双创（30/68 开头）和 ST 股，按连板数降序取前 N 只候选，然后对每只进行四维打分，按加权综合分排序输出 Top N。

四维判断体系
一、带动性（权重 35%）— "你是带头大哥吗？"
核心问题：这只股票封板时，板块里其他股票跟不跟？

板块共鸣（30%）：同行业涨停股占行业总股数的比例，>10% 满分
跟风力度（30%）：同行业非涨停股中涨幅 >3% 的比例，>15% 满分
封板决策力（40%）：在行业内的封板排名 + 封板时间的早晚 + 小弟跟进的紧密度（平均时间差越小越好）。一字板且换手率低会被乘 0.85 惩罚
每多一连板额外 +5 分（封顶 100）。取近 3 个涨停日中最佳一天的得分。

二、抗跌性（权重 15%）— "大盘崩了你扛得住吗？"
核心问题：大盘跳水（单日跌幅 < -0.7%）时，这只票是跟跌还是硬扛？

如果近期无跳水日，中性给 50 分。有跳水日时，每个跳水日做三维评估：

相对回撤（40%）：个股涨跌幅 - 大盘涨跌幅。超额收益越高越好，个股逆势收红 = 满分
日内承接（30%）：下影线比例 + 收盘位置，惩罚日内最大跌幅
反弹弹性（30%）：次日 alpha（个股次日涨幅 - 大盘次日涨幅），正 alpha 得分高
连续暴跌日如果个股跌得比大盘少一半，额外 +10。

三、领涨性（权重 25%）— "平时你在行业里排老几？"
核心问题：不看涨停日，平时这票在同行业里是不是领跑的？

当日真实排名：在行业成分股中按涨跌幅排序，找到这只票的确切分位
历史估算排名：用行业涨幅的中位数和标准差做正态分布近似，估算近 5 个非涨停日的排位
两者加权（当日 60% + 历史 40%），再加上偏离度加分（比行业中位数高出几个标准差）

四、资金承接性（权重 25%）— "别的板块的钱是不是跑你这来了？"
核心问题：市场恐慌时，其他板块的资金是不是虹吸到了这个板块？

用多板块的 5 分钟 K 线，滑动窗口 6 根 bar（=30 分钟），检测 "跨板块虹吸事件"，必须同时满足：

≥2 个其他板块窗口内跌 >1%
目标板块窗口内涨 >0.3%
6 根 bar 中至少 4 根阳线（方向一致）
到尾盘回撤 <30% 窗口涨幅（不是冲高回落，是真的留住钱了）
每个事件打分看三个维度：虹吸强度（40%）+ 广度/多少板块被抽血（20%）+ 持续性/回撤小（40%）。取最佳事件 + 多次事件加成。

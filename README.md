整体流程
从当日涨停榜出发，过滤掉双创（30/68 开头）和 ST 股，按连板数降序取前 N 只候选，然后对每只进行四维打分，按加权综合分排序输出 Top N。

四维判断体系
一、带动性（权重 35%）— "你是带头大哥吗？"
核心问题：这只股票封板时，板块里其他股票跟不跟？

板块共鸣（30%）：同行业涨停股占行业总股数的比例，>10% 满分
跟风力度（30%）：同行业非涨停股中涨幅 >3% 的比例，>15% 满分
封板决策力（40%）：在行业内的封板排名 + 封板时间的早晚 + 小弟跟进的紧密度（平均时间差越小越好）。一字板且换手率低会被乘 0.85 惩罚
每多一连板额外 +5 分（封顶 100）。取近 3 个涨停日中最佳一天的得分。
